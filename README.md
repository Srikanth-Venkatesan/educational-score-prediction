# Educational Score Prediction using IRT and ML Models

This repository implements multiple approaches for predicting student assessment scores:

## Implemented Models

- Linear Regression
- Polynomial Regression
- Ridge Regression
- Random Forest
- TF-IDF + Ridge
- TF-IDF + KNN
- Neighbour-based IRT (Expert Model)

## Evaluation Strategy

- Rolling-origin term evaluation
- RMSE
- MAD
- Question-level calibration diagnostics
- Difficulty vs Discrimination analysis
